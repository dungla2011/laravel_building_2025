Usage Examples
Basic Usage:
// Legacy compatible
showToast('success', 'Message');

// Advanced usage
toastSystem.success('Message', {
    title: 'Success',
    showProgress: true,
    delay: 3000
});
jQuery Usage:
$.toast.success('Message');
$.toast.error('Message', {delay: 0});
$.toast.clear();
